# Smoker Logger

An offshoot of the Lager_Logger V2 repo of mine, but using a MAX31855 thermocouple instead of a thermistor for temperature readings.
This project allows you to read from multiple thermocouples with a single ESP32 at the same time via webserver.
Designed a custom PCB with cute little smoker icons on it to simplify & tidy up the wiring.
